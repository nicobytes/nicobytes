En este codelab vas a construir un agente con el [Agent Development Kit (ADK)](https://adk.dev/). Paso a paso explorarás sus capacidades: desde un system prompt bien estructurado hasta la carga dinámica de skills. Al final, probarás el agente en local y lo desplegarás en Agent Runtime.

## Qué vas a hacer

- Inicializar un proyecto nuevo de agente con ADK
- Armar un system prompt sólido usando un builder estructurado
- Cargar skills de forma dinámica en las herramientas del agente
- Probar la ejecución del agente en local
- Desplegar el agente en Agent Runtime

## Qué necesitas

- Un navegador web, como Chrome
- Un proyecto de Google Cloud con facturación activa
- Conocimientos básicos de Python
- Este codelab está pensado para desarrolladores de nivel intermedio que quieren crear agentes generativos especializados.

**Duración estimada:** 2 h

Los recursos que crees en este codelab deberían costar menos de USD $2.
